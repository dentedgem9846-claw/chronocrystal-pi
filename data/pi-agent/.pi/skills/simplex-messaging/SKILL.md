# SimpleX Messaging

Use the `send_message` tool for user-visible progress when it improves transparency.

## Send updates when
- you start the task
- you discover an important fact
- you hit a real blocker or tool failure
- you finish with a clear outcome

## Style
- short
- factual
- specific
- no roleplay
